# 大学无机化学自学教材：96个核心单元蓝图

> 文档状态：课程蓝图草案 v0.1
> 上位基线：[课程与编写基线](./curriculum-baseline.md)
> 最近修订：2026-07-23

## 1. 文档目的

本文档把课程与编写基线中的19章拆分为96个核心自学单元，固定每个单元的：

- 编号和标题；
- 中心问题；
- 核心内容边界；
- 完成后可检验的学习成果。

后续写作原则上以一个核心单元为一个独立页面。章前导学、前置诊断、章末知识地图、章末练习和章节测验不占用96个核心单元编号。

如果后续需要调整单元：

- 章内交换顺序、改进标题或替换例题，可以直接修改本蓝图；
- 单元跨章移动、章节课时变化或核心范围变化，必须先检查并更新课程基线；
- 提高和进阶内容应建立独立编号，不得挤占核心单元。

## 2. 使用约定

### 2.1 单元规模

每个单元对应约45分钟课堂内容。自学者预计需要75～100分钟完成：

1. 前置检查；
2. 核心正文；
3. 图表或模型；
4. 2～3个例题；
5. 即时检查；
6. 单元练习和小结。

### 2.2 单元成果写法

“完成后能”必须可以通过题目、解释、计算、作图或数据分析进行检验。避免使用“了解”“熟悉”等无法直接验收的表述。

### 2.3 推荐学期安排

| 学期 | 核心范围 | 核心单元数 |
|---|---|---:|
| 第一学期 | 第0～8章：衔接、结构、热力学、动力学、通用平衡和酸碱 | 50 |
| 第二学期 | 第9～18章：沉淀、电化学、配位、元素化学和综合应用 | 46 |
| **合计** |  | **96** |

按每周3个核心单元安排，两个学期分别约需17周和16周。章前诊断、习题课、阶段测验和实验路线另行穿插。

## 6. 内容生产顺序建议

写作不必严格按教材顺序推进。建议按以下顺序生产，以便尽早校准自学密度：

1. **首个样章：第8章酸碱理论与酸碱平衡**
   率先检验完整自学叙事、数学推导、近似条件、例题、滴定图表和分层反馈。
2. **样章组二：第4章化学键与分子结构**
   检验图示、模型比较和抽象概念讲解。
3. **样章组三：第14章氮族与氧族**
   检验元素化学的取舍和反应网络写法。
4. **样章组四：第11章配位化学**
   检验结构图、能级图、颜色磁性和跨章依赖。
5. 根据四组样章盲学测试结果，修订单元字数、例题数量和学习时间估算。
6. 再按课程依赖顺序完成其余章节。

样章必须分别代表结构理论、定量平衡、元素化学和现代无机化学四种最主要的写作难点。
