# Material 3 全站视觉适配设计

## 目标

将大学无机化学自学教材从 Material for MkDocs 的默认 teal 主题，调整为以
石墨灰蓝为主色的 Material 3 阅读界面。改造覆盖桌面、窄屏、浅色与深色模式，
保持现有 Markdown 内容、导航树、搜索、MathJax 和页面链接不变。

## 范围与边界

本轮只调整 `content/stylesheets/extra.css` 和 `mkdocs.yml` 中主题色配置所需的
最小项；不增加自定义模板、不改变页面结构、不引入 JavaScript、不修改教材正文。
因此，Material for MkDocs 继续负责导航、搜索、目录和响应式行为，视觉规则集中于
一个可维护的样式入口。

## 视觉系统

浅色模式使用暖白背景和白色内容面，深色模式使用蓝灰黑背景和略亮的内容面。两种模式
共享以下语义令牌：

- 主色：石墨灰蓝，用于链接、当前导航项、主要按钮和键盘焦点；
- 容器色：比页面背景略深或略浅的低对比表面，用于卡片、提示框和表格表头；
- 描边：低对比中性灰蓝，用于分隔和输入框，避免重阴影；
- 圆角：导航及紧凑控件 10–12px，卡片、提示框和首页模块 14–16px；
- 阴影：仅用于浮层、搜索结果和首页卡片，使用短而淡的单层阴影；
- 交互：链接和按钮有可见 hover、active、focus-visible 状态；动效限定在 160–200ms，
  并尊重 `prefers-reduced-motion`。

## 组件覆盖

### 站点框架

顶部栏、标签导航、左右侧栏和页脚统一为分层 surface；当前页面和 hover 项使用浅色
容器与灰蓝文字，而非强烈的整块填色。搜索框、搜索结果、菜单和返回顶部按钮采用相同
圆角、描边、焦点环和阴影规则。

### 阅读页面

正文保持教材所需的高行距和受控行宽。标题强化字号、字重与间距层级；链接有可读的
下划线/悬停反馈。引用、admonition、details、表格、代码块、行内代码、任务清单、
脚注和目录锚点均使用统一的 surface 与描边，且窄屏下表格、代码和公式可横向滚动。

### 首页与操作控件

首页三列卡片改为 16px 圆角的表面卡片，在 hover 和键盘焦点时轻微抬升。主按钮使用
灰蓝填充，次按钮保持描边；两者具备足够触控尺寸与明确的禁用/焦点样式。

### 深色与窄屏

深色模式替换所有页面背景、容器、文字、描边与代码高亮相关变量，而不是仅反转正文。
宽度小于 60rem 时，首页卡片单列显示；侧栏、目录、搜索和表格沿用主题自带的响应式
行为，只调整间距、点击目标与视觉状态。

## 兼容性与验证

- 使用 CSS 自定义属性和 Material for MkDocs 已公开的 `--md-*` 变量，避免依赖脆弱的
  内部 DOM 结构；
- 不改变 Markdown 渲染结果，因此 MathJax、代码复制、搜索与导航瞬时切换仍按原逻辑运行；
- 构建使用 `mkdocs build --strict`；
- 在首页、一个长正文页和一个包含表格/提示框/代码的页面检查浅色、深色、桌面和窄屏；
- 核对正文、主色文字和焦点环的对比度，并使用键盘 Tab 检查主要交互控件。

## 不在本轮范围

不新增阅读进度、底部移动导航、账户功能、内容插图或自定义页面模板。后续如需其中任一
能力，应以独立设计与实现计划处理。
