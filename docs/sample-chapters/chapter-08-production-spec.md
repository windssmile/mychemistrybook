# 第8章样章制作规范：酸碱理论与酸碱平衡

> 文档状态：样章规范 v0.1
> 对应蓝图：第8章，8.1～8.6，共6个核心自学单元
> 目标读者：应用化学与理论化学方向大学一年级学生
> 最近修订：2026-07-23

## 1. 样章目的

第8章是本教材的第一个完整样章，用于验证：

- 没有教师口头补充时，正文能否独立完成概念建立；
- 前置诊断能否把学习者导向具体复习位置；
- 数学推导是否完整但不过度；
- 近似条件是否被明确说明和检验；
- 例题能否展示模型选择、计算、检查和化学意义；
- 即时检查与分层答案能否提供有效反馈；
- 图表能否帮助理解物种分布、缓冲和滴定；
- 核心、提高和进阶内容能否清楚分层；
- 单元实际学习时间是否符合75～100分钟的估算。

本章不是分析化学中的系统酸碱滴定课程，也不是物理化学中的严格溶液热力学。它负责建立大一无机化学使用的酸碱模型和水溶液平衡方法。

## 2. 章节中心问题

本章围绕一条连续问题链展开：

```text
什么是酸和碱？
→ 酸碱性为什么依赖结构和溶剂？
→ 水中各种酸碱物种有多少？
→ 怎样建立和近似求解酸碱平衡？
→ 多步反应和盐水解怎样处理？
→ 缓冲体系为什么稳定？
→ 滴定过程中主导关系怎样切换？
```

第8章结束时，学习者应从“套pH公式”提升到：

> 先识别物种和反应，再选择模型，建立平衡与守恒关系，判断合理近似，计算并检查结果。

10. 单位和数量级。

诊断结果应按知识点给出复习链接，不给“基础较差”之类笼统标签。

## 6. 数学与近似规范

### 6.1 推导原则

所有公式必须从反应、平衡常数和守恒关系开始。不得直接给出“弱酸公式”“缓冲公式”而不说明来源。

标准顺序：

```text
写反应
→ 列初始组成
→ 定义变化量
→ 写平衡组成
→ 写平衡常数
→ 判断能否近似
→ 求解
→ 反代验证
→ 检查化学意义
```

### 6.2 活度与浓度

- 第7章已经介绍活度是严格平衡量；
- 第8章核心计算可在稀溶液教学条件下使用浓度近似；
- 第一次使用时必须标注这是近似；
- 高浓度、较大离子强度或精密计算的修正属于提高内容；
- 不应把无量纲平衡常数和带单位的浓度商混为一谈。

### 6.3 近似

不能把“5%规则”写成自然定律。应说明它只是教学中判断某项忽略是否造成可接受误差的经验标准。

每次近似至少检查：

- 忽略量与保留量之比；
- 结果是否满足物料和电荷守恒；
- 浓度是否非负；
- pH是否处于合理范围；
- 主要物种判断是否与计算一致。

## 7. 符号和术语

全章统一使用：

| 项目 | 约定 |
|---|---|
| 酸解离常数 | \(K_\mathrm{a}\) |
| 碱解离常数 | \(K_\mathrm{b}\) |
| 水的自偶电离常数 | \(K_\mathrm{w}\) |
| 负对数 | \(\mathrm{p}K=-\log_{10}K\) |
| 活度 | \(a_i\) |
| 物质的量浓度 | \(c_i\) 或 \([i]\)，首次使用时说明近似 |
| 分析浓度 | \(c_\mathrm{A}\)、\(c_\mathrm{B}\) 等 |
| 氢离子水溶液简写 | \(\ce{H3O+}\)；注明真实水合结构更复杂 |
| 共轭酸碱对 | 相差一个\(\ce{H+}\)且电荷相差一个单位 |

正文以“质子”帮助初学者理解；术语表说明IUPAC在精确定义中使用 hydron（氢核的统称），而 proton 严格指 \(^1\mathrm{H}^+\)。

## 13. 样章验收

第8章样章只有满足以下条件才算完成：

- 六个核心单元全部完成；
- 章前诊断可以把错误映射到具体前置小节；
- 单元之间不重复主讲同一内容；
- 所有公式有来源、变量、条件和近似说明；
- 所有近似都经过反代或误差检查；
- 所有图有教学目的、图注和文字替代；
- 每单元至少两个完整例题和一个迁移问题；
- 章末练习覆盖模型、计算、图表和解释；
- 练习有提示、思路和完整答案；
- 术语、符号、数据条件全章一致；
- 没有提前依赖第9章沉淀或第11章配位平衡；
- 至少一名符合目标读者基础的学习者完成盲学测试；
- 根据盲学记录修正停顿点、错误路径和预计时间。

## 14. 制作顺序

1. 完成8.1正文，以确定叙事、折叠答案和术语格式；
2. 完成8.3正文，以确定计算、近似和例题格式；
3. 完成8.6正文，以确定图表和分段求解格式；
4. 回填8.2、8.4、8.5；
5. 制作章前诊断和章末知识地图；
6. 制作章节练习、测验和分层答案；
7. 完成科学审校和盲学测试；
8. 再决定是否把该模板推广到其余章节。
