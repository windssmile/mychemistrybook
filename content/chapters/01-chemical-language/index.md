# 第1章 化学语言、化学计量、气体与溶液

## 本章中心问题

怎样用名称、化学式、方程式和物理量准确描述物质与反应？怎样把微观粒子、
可测质量、气体状态和溶液组成连接起来？

## 学习本章之前

### 必须掌握

- 初中阶段的元素、简单化学式和方程式；
- 比例、百分数和一元方程；
- [第0章的课程地图与自学系统](../00-orientation/chapter-review.md)。

### 建议熟悉

- 高中阶段物质的量、气体和溶液基础；
- 科学计数法和单位换算。

### 3分钟诊断

1. \(\ce{Na+}\)与\(\ce{SO4^2-}\)形成电中性化合物时，数目比是多少？
2. \(36.0\ \mathrm{g}\)水是多少摩尔？取
   \(M(\ce{H2O})=18.0\ \mathrm{g\,mol^{-1}}\)。
3. 配平\(\ce{H2 + O2 -> H2O}\)。
4. 将\(250\ \mathrm{mL}\)换算为升。

<details markdown="1">
<summary>查看诊断答案</summary>

1. \(2:1\)，化学式为\(\ce{Na2SO4}\)；
2. \(2.00\ \mathrm{mol}\)；
3. \(\ce{2H2 + O2 -> 2H2O}\)；
4. \(0.250\ \mathrm{L}\)。

答题困难并不阻止开始本章。第1题由1.1补给，第2题由1.2补给，第3题由1.3补给，
第4题需要先复习单位前缀，并会在1.5和1.6反复练习。

</details>

## 本章学习成果

完成本章后，你应该能够：

1. 在常见无机物名称、化学式、离子组成和电荷之间转换；
2. 在粒子数、物质的量、质量和组成之间转换；
3. 配平基础反应并解释计量系数；
4. 处理限量试剂、纯度和产率问题；
5. 使用理想气体方程和分压关系；
6. 表示溶液组成并完成配制、稀释与混合计算；
7. 书写基础净离子方程式；
8. 使用单位、有效数字、守恒和数量级检查答案。
