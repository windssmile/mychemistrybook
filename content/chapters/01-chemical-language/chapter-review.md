# 第1章复习：从化学式到可测数量

## 章末任务

第1章建立了后续所有定量章节共同使用的语言。真正的章末成果不是记住六组公式，
而是能把一个问题沿着正确路线拆开：

```text
对象与化学式
→ 指定计数实体
→ 质量、物质的量、粒子数
→ 配平反应与计量系数
→ 限量试剂与反应进度
→ 气体状态或溶液组成
→ 单位、守恒、数量级和模型边界检查
```

请先不看各单元答案完成本页，再按错误位置返回具体小节。

---

## 1. 全章知识地图

```text
化学表示
├─ 元素、原子、单质、化合物、混合物
├─ 分子、离子、化学式单位
├─ 下标、系数、电荷、状态
└─ 名称 ↔ 化学式 ↔ 组成

数量桥梁
├─ N = nN_A
├─ M = m/n
├─ c = n/V
└─ pV = nRT

化学反应
├─ 写对物种与状态
├─ 配平元素与电荷
├─ 系数给出物质的量比
├─ Δn_i = ν_iΔξ
└─ min(n_i/|ν_i|) → ξ_max

实际样品
├─ 纯度 → 有效反应物
├─ 限量试剂 → 理论产量
└─ 实际/理论 → 百分产率

气体
├─ 状态变化
├─ 密度与摩尔质量
├─ 分压与物质的量分数
└─ 水上集气修正

溶液
├─ 配制、稀释、混合
├─ 分析浓度与物种浓度
├─ 电解质与离子计量
└─ 分子、完整离子、净离子方程式
```

## 2. 核心符号与单位

| 量 | 符号 | 常用单位 | 首要检查 |
|---|---|---|---|
| 实体数 | \(N\) | 1 | 实体是否明确 |
| 物质的量 | \(n\) | \(\mathrm{mol}\) | 不是质量 |
| 阿伏伽德罗常数 | \(N_\mathrm{A}\) | \(\mathrm{mol^{-1}}\) | 精确值不决定实验精度 |
| 质量 | \(m\) | \(\mathrm{g}\)、\(\mathrm{kg}\) | 与摩尔质量单位配套 |
| 摩尔质量 | \(M\) | \(\mathrm{g\,mol^{-1}}\) | 区分相对质量 |
| 反应进度 | \(\xi\) | \(\mathrm{mol}\) | 必须对应指定方程式 |
| 化学计量数 | \(\nu_i\) | 1 | 反应物负、产物正 |
| 压强 | \(p\) | \(\mathrm{kPa}\)、\(\mathrm{Pa}\) | 与\(R\)配套 |
| 体积 | \(V\) | \(\mathrm{L}\)、\(\mathrm{m^3}\) | 是气体还是溶液体积 |
| 热力学温度 | \(T\) | \(\mathrm{K}\) | 不直接代摄氏温度 |
| 物质的量浓度 | \(c\) | \(\mathrm{mol\,L^{-1}}\) | 分母是最终溶液体积 |
| 质量分数 | \(w\) | 1或% | 分母是混合物总质量 |

## 3. 看到题目后怎样选择路线

### 3.1 没有反应

```text
问粒子数？
→ N = nN_A

问质量或摩尔质量？
→ M = m/n

问气体状态？
→ pV = nRT

问溶液浓度？
→ c = n/V
```

### 3.2 发生反应

```text
写对物种与状态
→ 配平方程式
→ 把所有反应物转换为mol
→ 比较 n_i/|ν_i|
→ 找ξ_max与限量试剂
→ 求理论产量或剩余量
→ 再转质量、气体体积或溶液浓度
→ 若给实际值，再求产率
```

### 3.3 水溶液混合

```text
先问是否发生反应
├─ 无：合并同一组分n，再除以V最终
└─ 有：先做限量与计量
       → 再除以V最终
       → 写净离子方程式并查电荷
```

## 4. 综合练习

### 综合练习1：从化学式到组成

对\(\ce{Al2(SO4)3 * 18H2O}\)：

1. 说明它是分子式还是水合离子固体的化学式；
2. 一个化学式单位中Al、S、O、H原子计数各是多少；
3. \(0.0200\ \mathrm{mol}\)该物质含多少摩尔S原子和水合部分的水分子；
4. 为什么不能把点号后的水简单解释成称量时表面沾到的任意水？

### 综合练习2：纯度、限量试剂、气体与产率

石灰石样品质量为\(10.0\ \mathrm{g}\)，其中\(\ce{CaCO3}\)质量分数为\(80.0\%\)。
加入\(100.0\ \mathrm{mL}\)、
\(1.00\ \mathrm{mol\,L^{-1}}\ \ce{HCl}\)。题设反应为：

\[
\ce{CaCO3(s) + 2HCl(aq)
-> CaCl2(aq) + CO2(g) + H2O(l)}
\]

取\(M(\ce{CaCO3})=100.1\ \mathrm{g\,mol^{-1}}\)。

实际在\(298.15\ \mathrm{K}\)水上收集到
\(1.000\ \mathrm{L}\)湿气体，总压\(100.0\ \mathrm{kPa}\)，水蒸气压
\(3.17\ \mathrm{kPa}\)。取
\(R=8.314\ \mathrm{kPa\,L\,mol^{-1}\,K^{-1}}\)。
假设气相只含\(\ce{CO2}\)和水蒸气。

求：

1. 限量试剂；
2. \(\ce{CO2}\)理论物质的量；
3. 干\(\ce{CO2}\)实际物质的量；
4. 以气体物质的量计的百分产率；
5. 理论模型下剩余\(\ce{CaCO3}\)质量。

### 综合练习3：混合溶液与电荷检查

混合：

- \(40.0\ \mathrm{mL}\)、
  \(0.150\ \mathrm{mol\,L^{-1}}\ \ce{AgNO3}\)；
- \(50.0\ \mathrm{mL}\)、
  \(0.0800\ \mathrm{mol\,L^{-1}}\ \ce{NaCl}\)。

题设认为\(\ce{AgCl}\)完全按计量沉淀，最终体积为\(90.0\ \mathrm{mL}\)。

1. 写净离子方程式；
2. 求沉淀理论物质的量；
3. 求剩余过量离子的计量浓度；
4. 求旁观离子浓度；
5. 检查反应后溶液总正、负电荷浓度是否相等。

### 综合练习4：错误链定位

学生计算某气体反应时写出：

```text
25 °C代入T=25
→ 使用R=8.314 kPa·L·mol⁻¹·K⁻¹但p用atm
→ 由未配平方程式直接取系数比
→ 选择质量较小的反应物为限量试剂
→ 实际产量除以初始反应物质量得到“产率”
```

逐项指出错误，并给出正确检查顺序。

## 5. 综合练习答案

### 综合练习1

<details markdown="1">
<summary>提示</summary>

点号前后的组成都要计数；\(\ce{SO4}\)中的O与18个水中的O分别贡献。

</details>

<details markdown="1">
<summary>完整答案</summary>

这是水合离子固体的化学式，不宜把完整晶体简单称为独立分子。

每个化学式单位：

- Al：2；
- S：3；
- O：硫酸根部分\(3\times4=12\)，水合部分18，共30；
- H：\(18\times2=36\)。

\(0.0200\ \mathrm{mol}\)该物质含：

\[
n(\ce{S})=3(0.0200)=0.0600\ \mathrm{mol}
\]

\[
n(\ce{H2O,\ 水合部分})=
18(0.0200)=0.360\ \mathrm{mol}
\]

点号记录确定的化学计量组成；表面任意附着水属于样品状态或杂质问题，不应写入
理想化学式。

</details>

### 综合练习2

<details markdown="1">
<summary>思路</summary>

按“纯度→两种反应物物质的量→除系数找限量→理论气体→扣水蒸气求实际干气体→产率”
顺序计算。

</details>

<details markdown="1">
<summary>完整答案</summary>

纯\(\ce{CaCO3}\)质量：

\[
m=(10.0)(0.800)=8.00\ \mathrm{g}
\]

\[
n(\ce{CaCO3})=
\frac{8.00}{100.1}
=0.0799\ \mathrm{mol}
\]

\[
n(\ce{HCl})=
(1.00)(0.1000)
=0.100\ \mathrm{mol}
\]

比较反应进度：

\[
\frac{0.0799}{1}=0.0799,\qquad
\frac{0.100}{2}=0.0500
\]

\(\ce{HCl}\)是限量试剂：

\[
\xi_\max=0.0500\ \mathrm{mol}
\]

\[
n_\text{理论}(\ce{CO2})
=0.0500\ \mathrm{mol}
\]

实际干\(\ce{CO2}\)分压：

\[
p_{\ce{CO2}}=100.0-3.17
\approx96.8\ \mathrm{kPa}
\]

\[
n_\text{实际}(\ce{CO2})=
\frac{(96.8)(1.000)}
{(8.314)(298.15)}
=0.0391\ \mathrm{mol}
\]

百分产率：

\[
\frac{0.0391}{0.0500}\times100\%
\approx78.1\%
\]

理论消耗\(\ce{CaCO3}\)为\(0.0500\ \mathrm{mol}\)，剩余：

\[
n_\text{剩余}=
0.0799-0.0500
=0.0299\ \mathrm{mol}
\]

\[
m_\text{剩余}=
(0.0299)(100.1)
=2.99\ \mathrm{g}
\]

样品原有\(2.00\ \mathrm{g}\)惰性杂质不计入该剩余\(\ce{CaCO3}\)质量。

</details>

### 综合练习3

<details markdown="1">
<summary>完整答案</summary>

净离子方程式：

\[
\ce{Ag+(aq) + Cl-(aq) -> AgCl(s)}
\]

初始量：

\[
n(\ce{Ag+})=(0.150)(0.0400)
=0.00600\ \mathrm{mol}
\]

\[
n(\ce{Cl-})=(0.0800)(0.0500)
=0.00400\ \mathrm{mol}
\]

\(\ce{Cl-}\)限量：

\[
n(\ce{AgCl})=0.00400\ \mathrm{mol}
\]

\[
n_\text{剩余}(\ce{Ag+})=
0.00600-0.00400
=0.00200\ \mathrm{mol}
\]

最终体积\(0.0900\ \mathrm{L}\)：

\[
[\ce{Ag+}]_\text{计量}
=\frac{0.00200}{0.0900}
=0.0222\ \mathrm{mol\,L^{-1}}
\]

\[
[\ce{Na+}]_\text{计量}
=\frac{0.00400}{0.0900}
=0.0444\ \mathrm{mol\,L^{-1}}
\]

\[
[\ce{NO3-}]_\text{计量}
=\frac{0.00600}{0.0900}
=0.0667\ \mathrm{mol\,L^{-1}}
\]

正电荷浓度：

\[
0.0222+0.0444=0.0666\approx0.0667
\]

与硝酸根负电荷浓度在舍入误差内一致。

</details>

### 综合练习4

<details markdown="1">
<summary>完整答案</summary>

错误依次为：

1. 气体方程必须使用\(298.15\ \mathrm{K}\)，不是25；
2. 若\(R\)用\(\mathrm{kPa\,L}\)，压强必须换成\(\mathrm{kPa}\)，或改用与atm配套的\(R\)；
3. 方程式必须先配平，系数才给出有效计量比；
4. 限量试剂应比较\(n_i/|\nu_i|\)，不能比较质量大小；
5. 百分产率是目标产物实际量除以同一目标产物理论量。

正确顺序：

```text
核对物种、状态和单位
→ 配平方程式并查元素/电荷
→ 把各反应物转成mol
→ 比较n_i/|ν_i|
→ 求理论产物
→ 按需要使用气体状态或溶液体积
→ 实际目标产物/理论目标产物
→ 做数量级、守恒与边界检查
```

</details>

## 6. 章节测验

1. 区分\(\ce{2H2O}\)中的系数2和\(\ce{H2O2}\)中的下标2。
2. 为什么固体\(\ce{NaCl}\)通常计数化学式单位而不是分子？
3. 一摩尔\(\ce{Al2(SO4)3}\)含多少摩尔O原子？
4. 写出质量、物质的量和粒子数之间的两个基本关系。
5. 配平\(\ce{NH3 + O2 -> N2 + H2O}\)，并检查。
6. 为什么系数比不是质量比？
7. 对\(\ce{2A + 3B -> C}\)，怎样由\(n_\mathrm{A}\)和\(n_\mathrm{B}\)判断限量试剂？
8. 区分纯度、理论产量、实际产量和百分产率。
9. 理想气体方程中为什么必须使用开尔文？
10. 写出\(p_i\)、\(x_i\)和\(p_\text{总}\)的关系。
11. 物质的量浓度分母是溶剂体积还是溶液体积？
12. 为什么分析浓度不一定等于某个具体物种的平衡浓度？
13. 写出\(\ce{Ba^2+}\)与\(\ce{SO4^2-}\)生成沉淀的净离子方程式。
14. 混合两种会反应的溶液后，为什么不能直接把初始物质的量除以总体积？

<details markdown="1">
<summary>查看测验要点</summary>

1. 系数改变计数单位数，下标改变物质组成和身份；
2. 离子固体是延展晶格，式子给最简离子比；
3. \(12\ \mathrm{mol}\)O原子；
4. \(N=nN_\mathrm{A}\)，\(M=m/n\)；
5. \(\ce{4NH3 + 3O2 -> 2N2 + 6H2O}\)；
6. 质量还要乘各物质摩尔质量；
7. 比较\(n_\mathrm{A}/2\)与\(n_\mathrm{B}/3\)，较小者限制；
8. 纯度描述样品组成；理论产量由计量上限给出；实际产量来自测量；百分产率为实际/理论；
9. 状态方程使用从绝对零点起算的热力学温度；
10. \(p_i=x_i p_\text{总}\)；
11. 最终溶液体积；
12. 组分可能因反应或平衡分布到多个物种；
13. \(\ce{Ba^2+(aq) + SO4^2-(aq) -> BaSO4(s)}\)；
14. 反应先改变物质的量，必须先做限量与计量。

</details>
