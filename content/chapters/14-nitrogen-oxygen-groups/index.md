# 第14章 氮族与氧族

## 本章中心问题

为什么\(\ce{N2}\)、\(\ce{O2}\)以强多重键小分子存在，而P、S更常形成单键笼、环、
链或网络？为什么同一元素能从负氧化态跨到高正氧化态，并随介质表现氧化性或还原性？
含氧酸的酸性、氧化性和脱水缩合又为什么不能由元素氧化态一项决定？

```text
价电子与第二周期强多重键
→ 单质和氢化物结构

多氧化态
→ 氧化物、含氧酸及其盐

酸碱—氧化还原—配位耦合
→ 介质选择优势物种与反应方向

分子到环境循环
→ 氮、氧、硫物种的工业与环境边界
```

## 本章学习成果

完成本章后，你应该能够：

1. 写出第15、16族价电子通式和主要氧化态；
2. 比较第二周期与较重同族元素的多重键和成链能力；
3. 比较两族氢化物的结构、热稳定性、酸碱性和还原性；
4. 用结构和能量解释\(\ce{N2}\)稳定与固氮困难；
5. 说明\(\ce{NH3/NH4+}\)的酸碱、配位和盐类反应；
6. 构建氮化物、氨和氮单质的基础转化网络；
7. 判断主要氮氧化物的结构、氧化态和磁性；
8. 比较亚硝酸/硝酸及其盐的酸碱与氧化还原行为；
9. 以物种和过程而非口号描述氮循环；
10. 比较N/P成键、磷同素异形体和磷含氧酸；
11. 用条件性惰性电子对分析As、Sb、Bi氧化态；
12. 用MO解释\(\ce{O2}\)顺磁性并比较\(\ce{O2/O3}\)；
13. 区分氧化物、过氧化物、超氧化物和活性氧物种；
14. 比较硫同素异形体、硫化氢/硫化物和多硫化物；
15. 构建\(\ce{S(-II)->S(0)->S(IV)->S(VI)}\)反应网络；
16. 比较\(\ce{SO2/SO3}\)、亚硫酸盐/硫酸盐的结构与反应；
17. 在含氧酸问题中分开酸性、氧化性、热稳定性和脱水缩合；
18. 识别工业与环境事实的系统边界和安全限制。

## 本章不负责

- 重讲Lewis、MO、酸碱平衡和电极电势；
- 穷举所有氮磷硫含氧酸、缩合酸和盐；
- 用空d轨道杂化作为较重p区高配位的通用解释；
- 系统讲工业装置、大气反应机理、生物代谢或医学处置；
- 用“氧化态高”直接推出强氧化剂，或用“含H多”直接推出多元强酸。

本章坚持“结构—物种—介质—反应”的顺序，以少数代表体系建立可迁移网络。
