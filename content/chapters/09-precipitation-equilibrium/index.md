# 第9章 沉淀—溶解平衡

## 本章中心问题

“难溶”究竟有多难溶？混合两种溶液后如何判断是否出现沉淀？同离子、pH和副反应
怎样改变溶解度？多种离子共存时能否选择性沉淀其中一种？一种沉淀为什么可能转化为
另一种，或在改变条件后重新溶解？

本章把第7章的通用平衡框架用于固—液体系：

```text
固体与溶液是否达到饱和平衡？
→ Ksp与摩尔溶解度

当前离子组成是否要求析出或溶解？
→ Qsp与Ksp

环境是否改变自由离子浓度？
→ 同离子、pH与副反应

多个固相或多个离子竞争时发生什么？
→ 开始沉淀条件、选择性、转化与耦合
```

## 本章学习成果

完成本章后，你应该能够：

1. 区分溶解度、摩尔溶解度和溶度积；
2. 用活度写难溶离子固体的\(K_\mathrm{sp}\)；
3. 按溶解计量关系在\(K_\mathrm{sp}\)与摩尔溶解度之间转换；
4. 说明为什么不同化学计量类型的\(K_\mathrm{sp}\)不能直接比较溶解度；
5. 用\(Q_\mathrm{sp}/K_\mathrm{sp}\)判断沉淀或溶解方向；
6. 计算同离子存在时的条件溶解度；
7. 分析阴离子质子化和金属水解造成的pH效应；
8. 计算开始沉淀的临界浓度；
9. 评价分步沉淀是否达到所需分离程度；
10. 用平衡常数组合判断沉淀转化；
11. 解释酸碱或给定副反应为何可促进溶解；
12. 区分热力学方向、成核动力学与实验观察；
13. 用物料守恒、电荷守恒和相存在条件检查答案；
14. 说明活度、离子强度和亚稳过饱和的模型边界。
