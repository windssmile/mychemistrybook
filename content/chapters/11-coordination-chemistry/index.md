# 第11章 配位化学基础

## 本章中心问题

配合物由哪些部分组成，怎样准确命名？相同组成为什么可能有不同几何与异构体？配体
怎样改变金属离子的分布、沉淀和氧化还原行为？软硬匹配能解释哪些选择性，又不能替代
哪些数据？配体场怎样分裂d轨道并影响自旋、稳定性、颜色与磁性？

```text
组成与语言
→ 中心、配体、齿数、配位数、氧化态和命名

空间结构
→ 几何、连接方式与立体异构

溶液分布
→ 逐级/总形成常数、竞争配位与螯合

经验选择性
→ HSAB及其边界

电子结构
→ 晶体场分裂、自旋、CFSE、颜色与磁性
```

## 本章学习成果

完成本章后，你应该能够：

1. 识别配位实体、中心原子/离子和配体；
2. 区分配位数、配体数和齿数；
3. 由化学式求配位实体电荷与金属氧化态；
4. 完成常见单核配合物的基础命名与写式；
5. 由金属族号和氧化态确定常用d电子数；
6. 判断常见配位数对应的可能几何；
7. 识别连接异构、几何异构和光学异构；
8. 使用逐级与总形成常数处理基础配位平衡；
9. 分析配体质子化、沉淀和竞争配位；
10. 解释螯合效应的熵、焓与预组织边界；
11. 用HSAB作定性选择性判断并说明局限；
12. 画出八面体、四面体和平面四方d轨道分裂；
13. 判断基础高/低自旋排布、CFSE和未成对电子数；
14. 定性解释配合物颜色与磁性；
15. 区分晶体场模型、价键语言与完整配体场/MO描述。
