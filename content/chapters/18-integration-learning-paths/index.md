# 第18章 综合问题、知识地图与后续学习

## 本章中心问题

遇到一个没有背过的分子、离子、配合物、固体或溶液反应网络时，怎样先识别对象，再
选择合适模型？怎样区分“热力学允许”“平衡上显著”“速率足够快”和“实验中确实
观察到”？酸碱、沉淀、配位与氧化还原同时发生时，怎样确定主反应并写出可检验的
约束？最后，怎样把实验数据组织成有证据、有条件、有边界的化学解释？

本章把全书方法压缩为：

\[
\boxed{
\text{识别对象与问题}
\rightarrow
\text{选择模型}
\rightarrow
\text{写出守恒与条件}
\rightarrow
\text{用数据检验}
\rightarrow
\text{说明结论强度和限制}
}
\]

## 本章定位

前17章主要回答“有哪些概念和典型化学”；本章回答“怎样把它们组合起来解决陌生
问题”。四个单元依次处理：

1. 从组成和电子结构预测几何、磁性、极性、颜色等性质；
2. 把反应方向、平衡组成、速率和路径分层；
3. 求解酸碱—沉淀—配位—氧化还原耦合体系；
4. 从谱图、滴定、动力学和电化学等数据形成受约束的解释。

## 学习成果

完成本章后，你应该能够：

1. 先判断研究对象是孤立分子/离子、配合物、溶液物种还是延展固体；
2. 在Lewis、VSEPR、分子轨道、配体场、晶格和能带等模型中作出有理由的选择；
3. 用电子计数、对称性、轨道占据和粒子间作用预测性质；
4. 分开判断反应的热力学方向、平衡限度、动力学快慢与实际路径；
5. 对耦合平衡先写物料/电荷守恒，再选择主反应和必要的平衡常数；
6. 使用条件常数或自由物种活度，避免重复计算同一效应；
7. 区分观察、由模型得到的推断以及尚未排除的替代解释；
8. 根据诊断结果规划物理化学、结构化学、配位化学、材料或生物无机等后续学习。

## 本章边界

本章不负责：

- 引入另一套未经前文建立的核心理论；
- 给出不看对象、介质和时间尺度的“万能解题口诀”；
- 仅凭颜色、气味或单一现象认定物质身份；
- 把相关性直接写成因果关系；
- 把\(\Delta_\mathrm rG<0\)直接写成“反应一定很快”；
- 用纸面推演替代未知物鉴定、危险性评估或规范实验。

若体系涉及强氧化剂、剧毒物、挥发性重金属化合物、放射性物质或未知混合物，应把
风险识别置于操作之前。
