# 大学无机化学，自学也能建立完整理解

这是一套面向**应用化学与理论化学方向大一学生**的开放自学教材。它从高中化学和初等数学出发，帮助你建立：

```text
组成与电子结构
→ 成键和空间结构
→ 能量与稳定性
→ 宏观性质
→ 反应方向、限度和速率
→ 元素及其化合物的规律
```
